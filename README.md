# Assignment 5: Data Structures and Strings in Python

This project demonstrates the use of basic data structures in Python.

## Task 1: Dictionary of Student Marks
The program stores student names and their marks using a dictionary.  
It accepts a student name as input and displays the corresponding marks.  
If the student name is not found, an appropriate message is shown.

## Task 2: List Slicing
The program creates a list of numbers from 1 to 10.  
It extracts the first five elements, reverses them, and displays both lists.
